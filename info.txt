¿Cómo funciona la API elegida?
- La API de OpenStreetMaps tiene tres APIs en conjunto: Overpass, que 
sirve para ver lugares cercanos a la ubicación del usuario, Nominatim, 
que sirve para buscar lugares, y Tiles, que es la que dibuja y retorna 
imagenes de 256x256px de distintas partes de los mapas. 

¿Qué datos entrega?
- Imágenes, ubicaciones y, si se utiliza además la Messaging API de 
OpenStreetMaps, se pueden retornar mensajes con sus metadatos. 

Funcionamiento de la documentación
- La documentación de OpenStreetMap (OSM) no funciona como la de una 
API comercial (como la de Google o Amazon), que suele ser un portal 
unificado con guías paso a paso. La documentación de OSM es 
descentralizada y comunitaria. 

La direccion de la API para retirarla es [https://api.openstreetmap.org/].
